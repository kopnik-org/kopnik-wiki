# O cieli Kopnik.org

**Kopnik.org je nekomerčná sociálna sieť (webová aplikácia) fungujúca na základoch Kopného práva. Cieľom Kopnik.org je vytvoriť jednoduchý nástroj na zjednotenie Slovanov po celom svete.**
